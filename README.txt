1. Go to https://github.com/RoseReyes/edamam
2. Click <> Code button and select CodeSpaces tab
3. Click on weary cobweb - this will open a visual studio IDE online 
4. When the IDE is fully loaded, click on the terminal tab below the main screen
5. It should display @RoseReyes ->/workspaces/edamam 
6. Type cd Edamam and hit enter - the directory url should now look like  @RoseReyes ->/workspaces/edamam/Edamam
7. Type in dotnet watch run - this will build and run the project. A pop-up should open on the bottom right corner of your screen
8. On the pop-up click Open in Browser
9. A new tab/page should open up with this url https://weary-cobweb-j47p5j4rgqpfqqvq-5130.app.github.dev/ - you should be able to see the app
